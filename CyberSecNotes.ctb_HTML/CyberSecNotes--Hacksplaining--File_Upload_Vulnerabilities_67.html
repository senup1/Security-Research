File Upload Vulnerabilities


File Upload Vulnerabilities occur when there isn't proper checking implemented for uploaded files.

Protection:

Store uploaded files on a seperate disk partition or cloud service

Make sure uploaded files are written without the executable flag in the file permissions

Implement checks server side instead of in client side as client side checks can be circumvented. For example: javascript checks can be prevented by loaded the page after blocking javascript

Rename uploaded files to make them difficult to locate
