What is a Large Language Model?


Many web LLM attacs rely on prompt injection. This is where an attacker uses crafted prompts to manipulate and LLM's output.

To detect LLM vulnerabilities:

  1. Identify LLM's inputs, direct (such as a prompt) and indirect (such as training data)
  2. Work out what data and APIs the LLM has access to
  3. Probe this new attack surface for vulnerabilities
