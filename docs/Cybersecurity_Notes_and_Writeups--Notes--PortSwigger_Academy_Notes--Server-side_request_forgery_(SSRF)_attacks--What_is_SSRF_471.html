What is SSRF?


Server-side request forgery is a web security vulnerability that allows an attacker to cause the server-side application to make requests to an unintended location.

The attacker may try to make a connection to internal-only services within the organization's infrastructure. In other cases, they may force the server to connect to arbitrary external systems and leak sensitivbe data.

Successful SSRF attacks can lead to the attacker maliciously using the newly accessed resources to attack external applications, leak sensitive information, or perform arbitrary command execution.
