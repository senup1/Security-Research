Processes


Every process has a unique process ID (PID)

To see active processes associated with the current terminal session use “ps”

ps does not use “-” for some flags. So using these flags would look like “ps a”

Here are some of its flags:

a: Displays all processes for all users

u: Provides a detailed format

x: includes processess not attatched to any terminal (includes system daemons that start at boot which show ? for TTY column)

m: dont use with aux, shows active threads

Here are the flags that require “-”

-e: seletcts every process on the system

-f: displays a full format listing

ps -ef is typically preferred over ps aux

With the “top” command you can see results in real time

Kill a process using “kill -9 <PID>”

To get detailed info on a process use cat /proc/<PID>/status
