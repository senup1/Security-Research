Corrupted file


This CTF provided a corrupted file stating a couple of bytes could be missing.

  1. First few characters indicate it is a jpeg as jfif is the standard compression format for jpeg: \x\FF\E0\00JFIF\00\00\00\00\00\00\FF\DB\00C\00
  2. Used online hexdump editor (https://hexed.it/) to open file
  3. Searched google for proper starting hex value for jpeg
  4. Adjusted incorrect 5C 78 hex values to correct FF D8 values
  5. File now opens properly and displays flag as text in image
  6. Converted image to text to get flag: picoCTF{r3st0r1ng_th3_by73s_1512b52a}
