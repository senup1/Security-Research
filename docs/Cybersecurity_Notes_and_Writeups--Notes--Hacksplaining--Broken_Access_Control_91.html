Broken Access Control


Example:

Having unlisted/not clearly indexed directories/files active and accessible

This can be called security through obscurity which is unsafe

Authentication is a good way to prevent BAC issues

A good access control scheme usually contains groups and policies. (like how linux has root, specific users, and all access customization)

Lightweight Directory Access Protocol (LDAP) is designed to make access control decisions very quickly but requires a large upfront investment
