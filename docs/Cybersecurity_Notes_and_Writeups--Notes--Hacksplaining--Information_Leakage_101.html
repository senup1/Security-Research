Information Leakage


Information Leakage occurs when your system unnecessarilky exposes information about the technology it runs on.

Protection:

Disable showing the web server you are using in HTTP headers. Or even better, misreport the web server.

Use clean URLs. Avoid paths ending in file extensions like .php, .asp, etc

Modify error messages to be very generic client side and only detailed on server side

Obfuscate javascript
