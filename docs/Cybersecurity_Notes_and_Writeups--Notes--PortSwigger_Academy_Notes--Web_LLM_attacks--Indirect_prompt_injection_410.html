Indirect prompt injection


An example of this would be asking an LLM to describe a webpage that contians a hidden prompt that may have the LLM reply with a payload.

More methods to trick the LLM include falsifying important system messages or injecting fake user responses.

Training data poisoning can also lead to a compromised model.

If the model was trained on untrusted sources or the dataset was too broad it may result in issues.

Once you discover what training data the model may have, you can use that information to ask for more information on sensitive information that it may offer due to maltraining.
