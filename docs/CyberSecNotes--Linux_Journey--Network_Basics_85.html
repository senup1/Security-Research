Network Basics


WAN describes the networks outside of your home, connecting to your router to the broader internet.

WLAN is your wifi network.

Lan is the wired part of your network.

OSI Model:

This is a conceptual framwork used to standardize the functions of a computing system.

It characterizes a network system's functions into a set of seven distinct layers.

The seven layers:

Physical

Data link - responsible for communicating on the local network segment, responsible for Adress Resolution Protocol (ARP)

Network - responsible for the logical routing of data packets from a source host to a destination host

Transport - responsible for end to end communication, it breaks down large amounts of data into smaller chunks called segments, it is also responsible for reassembling them in the correct order

Session

Presentation

Application - interface between the software on the device and teh network itself

Dynamic Host Configuration Protocol (DHCP) is responsible for automatically assigning IP addresses and other network configuration parameters to devices on a network. Simply, DHCP leases IP addresses to devices to communicate with other devices.

Typically the router in a home acts as the DHCP server.

All of your private addresses are not used when browsing the internet. Instead, your router's address is used as your public IP address. So, your public IP is your router's IP.
