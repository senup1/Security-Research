JavaScript prototypes and inheritance


An object in java is like a struct in some other languages. It is instantiated and accessed a bit differently.

Example:

const user = {

username: “temp”,

userId: 3453,

isAdmin: false

}

user.username //"temp"

user['userId'] //3453

The objects may also contain functions.

Ex:

const user = {

username: “blah”

exampleMethod: function(){

//whatever here

}

}

If you are referencing an object property, the JavaScript engine first tries to access this directly on the object itself. If an object doesn't have a matching property, then it is looked for in the object's prototype instead.

Objects inherit all properties of all its parents.

Typically browsers allow you to check the prototype of an object by checking its __proto__ property
