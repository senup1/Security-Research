Terminal tools:

exiftool (usage: exiftool <filename.fileextension>)

steghide (usage: steghide extract -sf <filename.fileextension> -p <password>)

strings (usage: strings <filename.fileextension>) (extracts and prints human readable text from non-text file)

hash-identifier (usage: hash-identifier) then paste in the hash and enter

hashcat (usage: hashcat -m (module number) (file containing hashes) (directory for list of bruteforce strings) some modules for hashcat are 0 for md5, 100 for SHA-1, 1400 for SHA-256

sha256sum (usage: sha256sum <filename.fileextension>) (decrypts from sha256sum

* allows you to apply something to every file in a directory in console
