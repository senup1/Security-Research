SQL injection UNION attacks


When an application that is vulnerable to SQLi and the results of the query are returned, you can use the UNION keyword to retrieve data from other tables and databases.

UNION enables you to execute one or more additional SELECT queries.

Example: SELECT a,b FROM table1 UNION SELECT c,d FROM table2

For UNION to work, two requirements must be met:

  • The individual queries must return the same number of columns
  • The data types in each column must be compatible between queries
