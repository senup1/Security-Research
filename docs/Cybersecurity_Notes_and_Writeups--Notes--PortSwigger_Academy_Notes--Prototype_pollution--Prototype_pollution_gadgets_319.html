Prototype pollution gadgets


A gadget provides a means of turning the prototype pollution vulnerability into an exploit.

This is any property that is:

  • used by the application in an unsafe way (without filtering/sanitization)
  • attacker-controllable via prototype pollution

In simple terms, if a parent object does not have a property set that a child object uses and it is able to be tampered with by a user, then it is possible to exploit any child object that requires that property.
