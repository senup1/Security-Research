SQL injection attack, querying the database type and version on MySQL and Microsoft


  1. Provided with storefront website and told to display database version
  2. Launched burpsuite and intercepted request of selecting random category
  3. Forwarded request to repeater and appended 'UNION+SELECT+NULL,+NULL--
  4. Internal error
  5. Tried appending 'UNION+SELECT+NULL,+NULL%23 which succeeded
  6. Moved on to testing which columns allow string by appending 'UNION+SELECT+'a',+NULL%23 and 'UNION+SELECT+'a',+'a'%23
  7. Discovered both columns return strings
  8. Appended 'UNION+SELECT+'a',%40%40version%23
  9. Successfully discovered version was 8.0.42-0ubuntu0.20.04.1 and completed lab:
