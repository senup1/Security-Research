Clickjacking


Clickjacking is tricking users into clicking on a harmful link by disguising it as something else.

An example:

Clone website with similar http address (t3sla instead of tesla)

Iframe original website into clone to appear legit

Cover entire page with transparent div

wrap div in a tag

make a tag do whatever you want

Clickjacking might not necessarily affect you but may affect your users.

Prevention:

Use the "Content-Security-Policy" HTTP header to prevent your website from being Iframed
