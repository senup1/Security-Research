SQL injection


SQL injection (SQLi) is a vulnerability that allows an attacker to interfere with the queries an application makes to its database.

This can allow an attacker to view data that they are not normally able to retrieve.

In many cases the attacker can modify or delete data.

Enumaration of SQLi can include:

  • sending a quote character like '
  • sending a conditional such as OR 1=1
  • sending a payload designed to trigger time delays
