File upload vulnerabilities


File upload vulnerabilities are when a web server allows the upload of files without enforcing any restrictions to prevent dangerous files.

Typically file upload vulnerabilities are present when developers try to impelement what they believe is a robust validation system that is either flawed or easy to bypass.

The worst case scenario is when a website allows you to upload server-side scripts.

These are capable of fully compromosing the control of the system after getting a web shell.

If a website accepts an image through a form and performs its valdiation by implicitly trusting the Content-Type header in the HTTP request, it can easily be circumvented.
