Weak Session ID's


Weak Session ID's can allow attackers to access accounts they shouldn't be able to without the correct login credentials.

If the session ID's being used for users are too short, increment obviously, are easily guessable then it will be easy for attackers to use short scripts to find accessible sessions.

Protection:

Use built in session management from modern frameworks

Sign your cookies to make them tamper proof

Use strong random numbers to generate session keys if not using a framework
