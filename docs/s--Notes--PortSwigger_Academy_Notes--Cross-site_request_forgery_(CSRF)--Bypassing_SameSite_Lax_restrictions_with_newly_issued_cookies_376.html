Bypassing SameSite Lax restrictions with newly issued cookies


Cookies with Lax SameSite restrictions can be sent in POST requests on some occasions.

An example of this is the default Chrome Lax not being enforced on POST requests for the first 120 seconds.

This is done in order to prevent breaking the functionality of SSO mechanisms.

Browsers only allow popup tabs after manual interaction from a user, so wrap any window.open() in window.onclick to count it as user interacted.
