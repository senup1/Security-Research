Mass Assignment


Mass Assignment vulnerabilities allow an attacker to update state they should not be permitted to change.

As an example:

Even if there is no set Admin button on a page where users can edit their username, pfp, etc, if the HTTP Request with the addition of a isAdmin: true header or something similar can be processed by the webserver, it can compromise the system

Protection:

Explicitly enumerate the data fields you wish to set from an HTTP request
