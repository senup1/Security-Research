Command Line


when changing directories:

/ in the beginning of the path always refers to root.

~ in the beginning of the path always refers to the home/user directory

cd - always takes you to the previous directory you were in

for ls:

-l is for long list

-a is for hidden files

-r is for reverse

file can be used to understand what a file actually is

Example: file test.jpg

Files with extensions may not actually have the contents expected of the extension

cat can be used to concatenate multiple files together (prints all contents)

cat can also be used to quickly write to a new file: cat > filename.txt, after clicking enter type the contents then Ctrl + D to save and exit

-n numbers all output lines (nl command does the same thing)

-b numbers all non-blank output lines

less can be used to view long files in a page view

use arrow keys to navigate page by page or line by line

use g to move to begging of file

use G to move to end of file

use h for more options

use q to quit

use / to search for text

use ? to search for text from end

after search use n and N to navigate occurences of searched text

after entering history use Ctrl + r to search for recently used command by text

alias:

making a temporary alias (lasts for the current terminal session): alias example='ls -la'

to make them permanent add them to ~/.bashrc

use unalias to remove temporary alias

grep -l can be used to grep the contents of files directly

head -n where n is any number displays the first n lines of a file

tail -n where n is any number displays the last n lines of a file

sort will alphabetically sort files

tr deletes specified characters. Example: tr -d “z”

uniq will remove duplicate adjacent lines

wc returns the number of lines, number of words, number of bytes in that order. Use -l, -w, -c for only the wanted info in that order

each line in the /etc/shadow file contains 9 fields seperated by colons. the second field is the hashed user password
