Finding columns with a useful data type


Typically the data you want will be in string form, so after determining the number of required columns, probe each column to test whether it can hold string data.

This can be done by submit a series of UNION SELECT payloads that place a string value in each column one at a time.

Example:

‘ UNION SELECT ’a', NULL, NULL, NULL--

‘ UNION SELECT NULL, ’a', NULL, NULL--

‘ UNION SELECT NULL, NULL, ’a', NULL--

‘ UNION SELECT NULL, NULL, NULL, ’a'--
