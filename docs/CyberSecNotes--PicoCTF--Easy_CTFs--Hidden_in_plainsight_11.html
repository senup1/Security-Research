Hidden in plainsight


This CTF provided an image and stated there was a hidden payload in the file.

  1. Ran exiftool on img.jpg
  2. Found base64 encrypted content in comment field
  3. Decoded using cyberchef to get text: steghide:cEF6endvcmQ=
  4. Decoded the additional encoded text in the result to get password for steghide image: pAzzword
  5. Used steghide extract -sf <file name> -p <password> to decode img.jpg
  6. Catted flag.txt (result of extraction) for flag: picoCTF{h1dd3n_1n_1m4g3_54e31417}
